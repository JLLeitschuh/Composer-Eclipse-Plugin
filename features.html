Composer Eclipse Plugin

Features

Composer Editor

Edit your composer.json with a dedicated editor, including autocomplete and drag & drop features

Search, install, update and remove dependencies

Manage your dependencies right from within eclipse, including search on packagist.org

Json Editor

Check the resulting composer.json any time on the composer.json tab

New Composer Project Wizard

Create and initialize new composer projects with a dedicated wizard. Will automaticall download composer.phar and initialize your composer.json with the correct settings

Inject psr-0 autoloader

Inject the require_once statement for the composer autoloader script in any file with the correct relative path to the vendor directory

Inject psr-0 namespace into new classes

Tired of copying namespace and use statments all over your project? Let the Composer inject the correct psr-0 namespace into new classes and interfaces

Choose how to execute composer commands

Either let the plugin download a composer.phar automatically into your project or use a global one, just select the corresponding option in the preferences

Automatically manage your buildpath

A composer enabled PHP project will automatically configure your buildpath according to your installed dependencies. This can improve the PDT indexing and build performance, as not the complete vendor directory is indexed. You can also set global inclusion/exclusion patterns which can come in handy to exclude Test folders from the buildpath.
